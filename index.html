HTML significa HyperText Markup Language (Linguagem de Marcação de HiperTexto).

  • O HTML é uma linguagem de marcação;
  • O HTML tem códigos para criar paginas na web;
  • O HTML é constituído de códigos que delimitam conteúdos específicos, segundo uma sintaxe própria.

CSS significa Cascading Style Sheets (Folha de Estilos em Cascata).

  • O CSS é a linguagem que usamos para estilizar um documento HTML;
  • O CSS descreve como os elementos HTML devem ser exibidos em páginas web;
  • O CSS economiza muito trabalho. Ele pode controlar o layout de várias páginas web de uma só vez.
